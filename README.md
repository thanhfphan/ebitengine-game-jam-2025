# Ebitengine Game Jam 2025

## Prerequisites

- [Go](https://go.dev/dl/) (version 1.24.4)
- [Taskfile](https://taskfile.dev/installation/) (optional)

## How to start

```bash
task run # requires taskfile
```

or

```bash
go run cmd/main.go
```
